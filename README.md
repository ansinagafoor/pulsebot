# Pulse Bot

A self-running Python bot that fetches weather information and a motivational quote, generates a daily summary, and runs automatically using GitHub Actions.

## Features

- Live weather updates from wttr.in
- Random motivational quotes from ZenQuotes
- Daily summary generation
- Automated execution using GitHub Actions

## Files

- `pulse_bot.py` - Main bot script
- `requirements.txt` - Required Python packages
- `daily.yml` - GitHub Actions workflow
- `summary.txt` - Generated output

## Technologies

- Python
- Requests
- GitHub Actions
- wttr.in API
- ZenQuotes API
